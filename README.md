### Development

Clone repo and:

```bash
npm install --global yarn
npm install --global expo-cli
yarn install
yarn start
```
Connect with [Expo Go for Android](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=en&gl=US) on the same local network.
